# COBankBackend

# 🚀 usage


## 📦 install dependencies

```bash
cd COBankFrontend
```

```bash
npm install
```

## 🏃‍♂️ run the app

```bash
npm run start-server --silent -- --quiet --trace-warnings
```
on other terminal

```bash
npm run start-server --silent -- --quiet --trace-warnings
```

## 🏃‍♂️ run the app in production mode

```bash
npm run build
```
